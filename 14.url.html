url(uniform resource locator)

웹 페이지의 주소값을 가리키는 말

url의 구성

scheme://prefix.domain:port/path/filemane(pageName)?param=value¶m=value...

  • scheme:// : 인터넷 서비스 유형(http, https)
  • /prefix : 도메인 접두어(www)
  • domain : 인터넷 주소명(naver.com) 또는 ip주소
  • port : 포트번호(http : 80m https : 443) 서비스
  • path : 서버의 경로(생략된 경우 사이트의 루트 디렉토리)
  • filemane : 페이지의 이름
  • param : 서버에 요청하는 속성
  • value : 서버에 요청하는 속성의 값
